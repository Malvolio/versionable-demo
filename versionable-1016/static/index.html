Demonstration of Versionable Concept

This site demonstrates multiple versions of the same app. The versions available are

  • January
  • February
  • March

For the purposes of demonstration, the different versions are available as different index files. In production, there would usually be only file available, named index.html and back-versions would be accessed through complicated direct URLs, like this: https://cdn.rawgit.com/Malvolio/versionable-demo/cdc7f6d/public/index.html
